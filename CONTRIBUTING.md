# Contributing to Transcend Rover

This project is currently in active solo development.
Contributions, suggestions, and issues are welcome once Phase 5 is complete.

## How to Report Issues

Open a GitHub Issue with:
- Which phase / package the issue is in
- Steps to reproduce
- ROS2 and Ubuntu version
- Any relevant terminal output

## How to Suggest Features

Open a GitHub Discussion or Issue tagged `enhancement`.
Check [`Docs/PHASES.md`](Docs/PHASES.md) first — your suggestion may already be on the roadmap.
